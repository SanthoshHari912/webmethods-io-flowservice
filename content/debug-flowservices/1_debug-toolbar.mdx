## Key shortcuts
When you hover on every control, the shortcut key for invoking the respective control is shown in tooltip.

 Action | Key shortcut
 ---| ---
 **Disable/Enable breakpoints**| CTRL + F8
 **Resume** | F8
 **Stop** | SHIFT + F5
 **Restart** | F5
 **Step over** | F10
 **Clear all breakpoints** | ALT + F8
 **Close debug panel** | CTRL + Q

 **Note** : If we invoke **Clear all breakpoints** action, we will not be able retrieve breakpoints back using undo action.
